
# Network Port Scanner

A beginner cybersecurity project built in Python.

## What it does
Scans a target IP or domain for open TCP ports 
and displays the service name for each port.

## Ports checked
| Port | Service |
|------|---------|
| 21   | FTP     |
| 22   | SSH     |
| 23   | Telnet  |
| 25   | SMTP    |
| 53   | DNS     |
| 80   | HTTP    |
| 443  | HTTPS   |

## How to run
python scanner.py

## What I learned
- How TCP connections work
- How sockets work in Python
- What open and closed ports mean in security
- How port scanning is used in real penetration testing
